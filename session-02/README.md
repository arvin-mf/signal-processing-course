## Sinyal Kontinu
**Sinyal Kontinu Periodik**

Sinyal periodik adalah sinyal yang memiliki pola berulang pada nilainya dalam panjang waktu tertentu.

Ekspresi umum sinyal sinusoidal kontinu:
$y=A\cdot sin(\omega t + \theta) $
